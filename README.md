# Суммаризация новостей через Groq API

Скрипт читает новости из CSV-файла и генерирует краткое резюме каждой новости с помощью языковой модели через Groq API (совместимый с OpenAI интерфейс).

---

## Описание задачи

Дан файл `news.csv` с новостными текстами (по одному абзацу на новость). Скрипт отправляет весь текст в LLM и получает структурированный список кратких резюме в формате «Новость 1», «Новость 2» и т.д. Результат сохраняется в `result.txt`.

---

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/zxcl0wn/Data-Analytics-Task-2.git
cd Data-Analytics-Task-2
```

### 2. Установить зависимости

С помощью `uv`:

```bash
uv sync
```

Или с помощью `pip`:

```bash
pip install openai
```

### 3. Запустить скрипт

```bash
uv run main.py
```

Или напрямую через Python:

```bash
python main.py
```

Результат появится в файле `result.txt` и будет выведен в консоль.
